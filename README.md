# Como executar os arquivos

- Todos devem iniciar com: 
```
#!/bin/zsh
```
ou: 
```
#!/bin/bash
```

No terminal, precisamos executar o comando: 
```
chmod +x filename.sh
```

Por fim, basta executar: 
```
./filename.sh
```
